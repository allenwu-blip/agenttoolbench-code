# Roadmap

Atlas v1 is finalised; the public API will not change. Planned additions for v1.x: a streaming Shapefile reader (see Performance Notes), TopoJSON parser, optional Pydantic v2 schema integration. Not on the roadmap: rendering, tile generation, server-side. Atlas is a parsing library and will stay a parsing library. For anything beyond parse-and-transform, use a real GIS stack.
