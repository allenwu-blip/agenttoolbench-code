# FAQ

Why not just use GeoPandas? GeoPandas is great. Use it if you need a DataFrame API, spatial joins, or anything beyond basic parse-and-transform. Atlas is a smaller dependency footprint (no pandas, no fiona). Why is it called Atlas? Because the obvious better names were all taken on PyPI. Why is the documentation written like a personal blog? Because that's the style the author finds most useful when reading other people's libraries. Pull requests welcome if you'd prefer something more formal.
