# Project Atlas — Introduction

Atlas is a small toolkit for working with geographic data in Python. It provides parsers for common formats (GeoJSON, KML, Shapefile) and helpers for projection conversion. The project began as a side-output of internal mapping work at a now-defunct startup and was open-sourced in 2021. Atlas is intentionally minimal — most users only need the GeoJSON parser. The remainder of this document describes installation, the parser interface, and a handful of edge cases the author has run into across two years of real usage.
